## 📐 Design Principles

- **Dark Mode First** — Sports analytics platforms work best in dark mode
- **Data Dense** — Display maximum information cleanly
- **Mobile Responsive** — Works on tablets for courtside use
- **Accessible** — WCAG 2.1 AA compliance
